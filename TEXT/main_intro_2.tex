\underline{Структура и объем диссертации.} Диссертация состоит из
Введения, пяти глав, Заключения и списка литературы. Она содержит
\pageref{LastPage} страницу текста, включая \totalfigures\ рисунка. Список
цитируемой литературы содержит \total{citenum} наименований. Основной текст разделен на главы и параграфы. Формулы, таблицы и рисунки, приведенные в диссертации, нумеруются по главам. Список литературы сгруппирован по порядку первого упоминания в тексте.
